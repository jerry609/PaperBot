<objective>
Make the MCP server (9 tools + 4 resources built in Phases 2-4) runnable by agents via stdio and Streamable HTTP transports, with a CLI command to start either mode.

Purpose: Without transport wiring, the MCP server singleton exists but cannot be reached by any agent. This plan connects it to the outside world.
Output: `serve.py` module, CLI `mcp serve` command, `[project.scripts]` entry, and comprehensive unit tests.
</objective>
