  <behavior>
    - test_fan_out_to_multiple_subscribers: append(event) delivers to all subscribed queues; two subscribers each receive the same event via queue.get_nowait()
    - test_ring_buffer_catch_up: append N events, then subscribe() -- new queue contains all N events (up to ring buffer max)
    - test_ring_buffer_maxlen: append 250 events (exceeds 200 default) then subscribe() -- queue contains only last 200
    - test_backpressure_drops_oldest: subscribe with queue maxsize=3, append 5 events -- queue contains last 3 (oldest 2 dropped)
    - test_unsubscribe_cleans_up: subscribe, unsubscribe, assert queue set is empty; append after unsubscribe does not raise
    - test_close_clears_all_queues: subscribe 3 clients, close() -- queue set is empty
    - test_append_accepts_dict: append({"type": "test"}) works without AgentEventEnvelope
    - test_append_accepts_envelope: append(AgentEventEnvelope) calls to_dict() and stores the dict
    - test_stream_returns_empty: stream("any_run_id") returns empty iterator (bus does not support historical replay)
    - test_protocol_compliance: isinstance(EventBusEventLog(), EventLogPort) is True
    - test_composite_includes_bus: CompositeEventLog([LoggingEventLog(), EventBusEventLog()]).append() fans out to bus subscribers
  </behavior>
  <implementation>
    Implement EventBusEventLog in src/paperbot/infrastructure/event_log/event_bus_event_log.py:

    - Class implements EventLogPort protocol
    - Constructor takes ring_buffer_size (default 200) and client_queue_size (default 256)
    - self._ring = collections.deque(maxlen=ring_buffer_size) stores serialized dicts
    - self._queues: Set[asyncio.Queue] = set() holds per-client queues
    - append(): convert event to dict (call to_dict() if AgentEventEnvelope, else dict(event)), store in ring, call _fan_out()
    - _fan_out(): iterate list(self._queues) snapshot, put_nowait() each; on QueueFull, get_nowait() oldest then put_nowait() new
    - subscribe(): create asyncio.Queue(maxsize=client_queue_size), drain ring buffer into it, add to _queues set, return queue
    - unsubscribe(q): discard q from _queues set
    - stream(): return iter(()) -- bus does not do historical replay
    - close(): clear _queues set

    Use put_nowait() only (append is sync def, not async). Iterate list(self._queues) snapshot to avoid set-mutation-during-iteration.

    Tests go in tests/unit/test_event_bus_event_log.py. Use @pytest.mark.asyncio for all async tests (strict mode). Use make_event() helper from message_schema to create test events. No mocking -- test the real class directly.
  </implementation>
</feature>
